### Features
- Multi host profile selection
- TFT display with status information
- OTA firmware update
- Desktop configuration and upload to device
- Macro mapping to single keys or actions
- LiPo batteries, high capacity, durable, rechargeable


### Communication
- ESP Now
- Bluetooth
- Bluetooth LE
- WiFi (TCP)
